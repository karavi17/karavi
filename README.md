# MangaKakalot Clone

A modern manga reading website built with React, TypeScript, and Tailwind CSS.

## Features

- **Home Page**: Browse popular and latest manga updates.
- **Search**: Find your favorite manga by title.
- **Manga Details**: View descriptions, genres, authors, and chapter lists.
- **Chapter Reader**: A clean, distraction-free reading experience with infinite scroll (vertical) support.
- **Responsive Design**: Optimized for both mobile and desktop.

## Getting Started

### 1. Start the API Server

This project depends on the `mangakakalot-api`. You can run it easily using npx:

```bash
npx mangakakalot-api
```

The API will start at `http://localhost:3000`.

### 2. Run the Frontend

```bash
cd manga-reader
npm install
npm run dev
```

The website will be available at `http://localhost:5173`.

## Tech Stack

- **Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router](https://reactrouter.com/)
- **API Client**: [Axios](https://axios-http.com/)
